Key Features

🔍

Deep Metadata Extraction

  • Sony XAVC Support: Automatically prioritizes XML sidecars (M01.XML) for camera model and date detection.
  • Auto-organizes media into YYYY-MM-DD / [Camera Model] / [Files] folder structures.
📊

Precision Transfer Engine

  • Byte-accurate ETA based on real MB/s throughput, not just file count.
  • Per-file progress bars + overall session tracking.
  • Intelligent folder suffixing (e.g., ILCE-7M3 (1)) to prevent collisions.
🌐

Advanced Automation

  • Remote Sync: Kicks off cloud/secondary backups automatically after ingestion.
  • Webhooks: Rich Discord/Slack notifications with session summaries.
  • Auto-Updates: Always stay on the latest version.
🎨

Premium Interface

  • Minimalist dark UI centered around a custom physical camera logo.
  • Bilingual support: English and Tiếng Việt.
